# Test fixtures

## test-image.jpg

A photograph of a laptop, mug, notepad, pen, and ruler on a desk.

- **Source:** [Wikimedia Commons](https://commons.wikimedia.org/wiki/File:Ruler_and_laptop_on_a_desk_(Unsplash).jpg)
- **Originally from:** [Unsplash](https://unsplash.com/)
- **License:** [CC0 1.0 Universal Public Domain Dedication](https://creativecommons.org/publicdomain/zero/1.0/)
- **Dimensions:** 640×427

This image is used for visual regression tests and the auto-detect smoke
test. It contains real texture (so blur is visibly different from the
original) and recognisable objects (laptop, cup, book) that the OWLv2
zero-shot detection model can find.
