# Gaussian Blur Tool (Work in progress)

A client-side image blur tool built with React, TypeScript, and Vite.

**[Demo](https://milesburton.github.io/gaussian-blur-tool/)**

## Sample images

The app ships with a small gallery of CC0 sample images so you can try
the tool without uploading your own. All samples are in
[`public/samples/`](public/samples/) — see
[`public/samples/README.md`](public/samples/README.md) for sources and
attribution. Every image is released under the
[Creative Commons CC0 1.0 Universal Public Domain Dedication](https://creativecommons.org/publicdomain/zero/1.0/),
sourced from Unsplash via Wikimedia Commons.
